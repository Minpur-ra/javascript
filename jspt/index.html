decalração de objetos em js

objeto é um tipo nao primitivo de dados, composto de propiedades e valores, a referencia voce nao muda, mas as propriedades voce muda ao decorrer desse código para declara objetos em js, você pode pegar ou um let, const ou var, criar uma variavel para ele e na frente colocar um sinal de igual e abrir chaves, em seguida crie as caracteristicas que quiser e atribue valores as mesmas ex: let aluno = { nome: "ka", nota: 7, matricula: 234 }

estruturas condicionais

condicional simples

estruturas condicionais servem para alterar o fluxo de execução do meu algoritmo após um teste de condição, se acondição for verdadeira, o fluxo de execução do código interno à estrutura condicional executa. caso contrário, o código salta para o final da estrutura condicional.

se (condição == verdadeira) {comandos serão executados} else{ comandos executados qnd for falso }
